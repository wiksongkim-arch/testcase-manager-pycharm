# 功能测试清单

## 测试环境准备

### 1. 安装插件
1. 运行 `./local-test.sh` (Linux/Mac) 或 `local-test.bat` (Windows)
2. 在 PyCharm 中安装生成的插件包
3. 重启 PyCharm

### 2. 准备测试文件
创建一个测试 YAML 文件 `test_cases.yaml`:

```yaml
test_cases:
  - id: TC001
    name: 登录成功测试
    priority: P0
    status: PUBLISHED
    steps:
      - 打开登录页面
      - 输入正确的用户名和密码
      - 点击登录按钮
    expected: 登录成功，跳转到首页
    tags:
      - login
      - smoke

  - id: TC002
    name: 登录失败测试
    priority: P1
    status: DRAFT
    steps:
      - 打开登录页面
      - 输入错误的密码
      - 点击登录按钮
    expected: 显示密码错误提示
    tags:
      - login

  - id: TC003
    name: 注册新用户
    priority: P2
    status: DRAFT
    steps:
      - 打开注册页面
      - 填写注册信息
      - 点击注册按钮
    expected: 注册成功
    tags:
      - register
```
